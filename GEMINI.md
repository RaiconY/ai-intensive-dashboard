## Метафора

- Ученики — кибер-агенты, убегающие от Blood Dragon
- Дракон движется автоматически (время идёт)
- Ученики двигаются когда выполняют задания
- Отстал — дракон догоняет, аватар в опасности

## Данные

JSON-файл `data/cohort-1.json` содержит:
- `cohort` — название потока
- `startDate`, `endDate` — даты потока (для позиции дракона)
- `students` — список учеников с аватарами
- `weeks[].sections[].tasks` — задания по неделям и секциям с очками
- `checkins` — выполненные задания по ученикам (объект: studentId → [taskId, ...])

## Редактирование

1. Открыть `/admin-x-y-z.html`
2. Ввести пароль
3. Кликать по ячейкам для toggle чекинов
4. Сохранить → данные отправляются в Netlify Blobs через `/api/save`

## Состояния аватаров

- **Fresh** (зелёный) — дракон далеко
- **Stressed** (оранжевый) — дракон приближается
- **Bitten** (красный) — дракон догнал
- **Leader** (золотой) — первое место

## Деплой

- Хостинг: Netlify (статика + Functions)
- Хранилище данных: Netlify Blobs
- Автодеплой при пуше в main
- Fallback: если API недоступен, дашборд читает локальный `data/cohort-1.json`
